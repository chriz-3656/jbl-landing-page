# JBL Tune 720BT - Premium Landing Page

This is a premium, cinematic 3D landing page for the JBL Tune 720BT headphones, built with Next.js, Three.js, React Three Fiber, GSAP, and Tailwind CSS.

## Features
- **3D Hero Experience:** Interactive 3D model of the JBL Tune 720BT headphones.
- **Cinematic Animations:** Smooth scroll-triggered animations using GSAP and Framer Motion.
- **Premium UI:** Dark luxury aesthetic, glassmorphism, and custom styling.
- **Responsive:** Fully optimized for mobile, tablet, and desktop viewing.

## Tech Stack
- [Next.js 15](https://nextjs.org/)
- [Three.js](https://threejs.org/) & [React Three Fiber](https://r3f.docs.pmnd.rs/)
- [GSAP](https://gsap.com/) & [Framer Motion](https://www.framer.com/motion/)
- [Lenis](https://lenis.studiofreight.com/) (Smooth Scrolling)
- [Tailwind CSS v4](https://tailwindcss.com/)

## Getting Started

1. Clone the repository
2. Install dependencies with `npm install --legacy-peer-deps`
3. Run the development server with `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)
